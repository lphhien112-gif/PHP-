PHP Lab02 — GET, POST, HEAD, OPTIONS

Khám phá sự kiện.
Đặt vé tức thì.

Nền tảng đặt vé sự kiện mini với REST API chuẩn, hỗ trợ đầy đủ HTTP methods và validation chuyên nghiệp.

🚀 Khám phá API ⚡ Thử ngay

Tại sao chọn EventHub?

Được xây dựng với kiến trúc MVC, PSR-4 autoloading và xử lý HTTP chuẩn REST.

📡

REST API Chuẩn

Hỗ trợ GET, POST, HEAD, OPTIONS với status codes chuẩn: 200, 201, 404, 405, 415, 422.

🔍

Filter & Search

Lọc sự kiện theo category, status thông qua query parameters. Kết quả trả về JSON chuẩn.

🛡️

Validation Chặt

Kiểm tra email, phone, required fields, integer, positive number. Trả lỗi chi tiết từng field.

API Endpoints

Click vào bất kỳ endpoint nào để thử nghiệm trực tiếp.

GET
/events
Lấy danh sách tất cả sự kiện. Hỗ trợ filter theo category & status.
200 OK 405 Method Not Allowed
GET
/events/{id}
Lấy chi tiết sự kiện theo ID. Trả 404 nếu không tìm thấy.
200 OK 404 Not Found
POST
/bookings
Đặt vé sự kiện. Yêu cầu JSON body với event_id, full_name, email, phone.
201 Created 415 Unsupported 422 Unprocessable
GET
/events?category=Technology
Lọc sự kiện theo danh mục công nghệ.
200 OK
